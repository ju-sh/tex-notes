\subsection{Vector space}

\begin{itemize}
\item A Set. Elements of this are called \emph{vectors}.
\item Elements can be added together
\item Any given element can be scaled by a scalar value
\item The scalars could be elements from a field
\item \emph{F-vector space}: scalars are from a field F
\end{itemize}

Definition of a vector space over a field $F$:
\begin{itemize}
\item A non-empty set $V$
\item A binary operation $+$: vector addition
\item A binary function $*$: scalar multiplication $(*) : F \to V \to V$
\end{itemize}

8 axioms:

\begin{itemize}
\item Addn is comm: $u + v = v + u$
\item Addn is assoc: $u + (v + w) = (u + v) + w$
\item Addn has idty ($0 \in V$): $v + 0 = v$
\item Addn has inverse ($-v \in V$): $v + (-v) = 0$
%
\item $k_1(k_2.v) = (k_1.k_2)v$
\item Mult has idty ($1 \in V$): $v.1 = v$
\item Mult distr over vec addn: $k(u + v) = ku + kv$
\item Mult distr over field addn: $(k_1 + k_2)v = k_1.v + k_2.v$
